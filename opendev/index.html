Viewing Source Code

  1. Open your browser. I will be using Safari.
  2. Pull up a site that you would like to inspect. I'm using the Wall Street Journal.
  3. On the webpage of your choice, right-click and choose 'Inspect Element'. On Mac, for Safari, you can also use the keyboard shortcut Option+Command+U.
  4. In doing so, you will have the Source Code come up on your screen.
